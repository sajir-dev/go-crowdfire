## Install and Setup
Go version => 1.16

1. Clone the repo
2. cd into the directory *cd go-crowdfire-win*
3. wait for packages to dowload automatically or enter *go get -u ./...*
4. *go run main.go*

*env-variables*
DB_HOST=localhost
DB_PORT=27017
DB_USERNAME=postgres
DB_PASSWORD=postgres
DB_NAME=crowdfire
